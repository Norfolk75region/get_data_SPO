# Описание
Скрипт для сбора данных из таблиц, доступных на [веб-платформе мониторинга образовательных организаций](https://monitoring.miccedu.ru/iam/2025/_spo/material.php?type=2&id=10606), за указанные годы. 
Он включает функции для получения данных, обработки таблиц и их сохранения в CSV-файл.
Для использования достаточно обновить список YEARS, дополнив необходимыми для выгрузки годами.

# Функции
## **`get_html_page(url)`**
Получает HTML-контент веб-страницы.
- **Параметры**:
    - `url` (str): URL веб-страницы для получения данных.

- **Возвращает**:
    - HTML-контент страницы (`str`), обработанный с помощью BeautifulSoup.

- **Пример использования**:
``` python
soup = get_html_page("https://example.com")
```
## **`get_organisation_id(year)`**
Извлекает список идентификаторов организаций за указанный год.
- **Параметры**:
    - `year` (int): Год, для которого нужно получить идентификаторы организаций.

- **Возвращает**:
    - Список идентификаторов (`list`), содержащих ссылки на страницах с информацией о каждом учреждении.

- **Пример использования**:
``` python
ids = get_organisation_id(2023)
```
## **`get_tables(url, id, year)`**
Собирает все таблицы с указанной страницы, преобразует их в DataFrame, добавляет данные о годе и идентификаторе учреждения.
- **Параметры**:
    - `url` (str): URL страницы с таблицами.
    - `id` (str): Идентификатор текущего учреждения.
    - `year` (int): Год, связанный с таблицами.

- **Возвращает**:
    - Список таблиц в формате `pandas.DataFrame`.

- **Пример использования**:
``` python
tables = get_tables("https://example.com", "123456", 2023)
```
## **`get_data_from_educations(educations, year)`**
Обрабатывает данные об организациях за конкретный год, фильтруя только нужные таблицы с соответствующими колонками.
- **Параметры**:
    - `educations` (list): Список образовательных организаций (или их идентификаторов).
    - `year` (int): Год, данные для которого обрабатываются.

- **Возвращает**:
    - Список DataFrame-таблиц с данными.

- **Пример использования**:
``` python
data = get_data_from_educations(["inst.php?id=123"], 2023)
```
## **`save_tables(data, year)`**
Сохраняет объединённые данные в один CSV-файл.
- **Параметры**:
    - `data` (list): Список DataFrame-таблиц, которые нужно сохранить.
    - `year` (int): Год, соответствующий сохранённым данным.

- **Сохраняет**:
    - CSV-файл в директорию `data` с именем `data_<year>.csv`.

- **Пример использования**:
``` python
save_tables(data, 2023)
```
## **`main()`**
Основная точка запуска программы. Автоматизирует процесс:
1. Указаны года `[2021, 2022, 2023, 2024]`.
2. Для каждого года:
    - Извлекаются идентификаторы организаций.
    - Список таблиц обрабатывается.
    - Данные сохраняются в CSV.

- **Пример запуска**:
``` python
if __name__ == "__main__":
    main()
```
# Зависимости
- **Библиотеки, используемые в скрипте**:
    - `requests`: для выполнения HTTP-запросов.
    - `BeautifulSoup` (из `bs4`): для парсинга HTML-контента.
    - `pandas`: для обработки табличных данных.
    - `pprint`: для красивого вывода данных в процессе отладки.

- **Внешние библиотеки**: Требуется установить библиотеки:
``` bash
  pip install requests beautifulsoup4 pandas
```
# Входные данные
- URL платформы мониторинга: `https://monitoring.miccedu.ru/iam/{year}`.
- Список годов для обработки: `[2021, 2022, 2023, 2024]`.

# Выходные данные
Для каждого года генерируется CSV-файл:
`data/data_<year>.csv`, содержащий объединённые таблицы с добавленными столбцами `year` и `id`.
# Ограничения
1. Скрипт предполагает наличие интернет-соединения для работы с платформой мониторинга.
2. Вероятна зависимость от структуры таблиц на платформе, которая может измениться.
